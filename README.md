## 📁 Structure

Problems are organized by topic:

| # | Topic | Description |
|---|-------|-------------|
| 01 | `arrays` | 1D array manipulation and traversal problems |
| 02 | `2darray` | Matrix and grid-based problems |
| 03 | `recursion` | Recursive thinking and backtracking |
| 04 | `hashmap` | Hashing, frequency maps, lookups |
| 05 | `sorting` | Comparison and non-comparison sorts |
| 06 | `pattern` | Number, star, and character patterns |
| 07 | `basic_math` | GCD, primes, digits, modular arithmetic |
| 08 | `stl` | Standard Template Library usage |
